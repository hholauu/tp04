En el mundo de la programación web, aprender a utilizar las tablas en HTML es una habilidad fundamental. Las tablas son una herramienta poderosa para organizar y presentar datos de manera estructurada en una página web. Permiten mostrar información de manera ordenada, crear diseños complejos y mejorar la accesibilidad de contenido.

Es importante recordar que el uso de tablas nos sirve para ORGANIZAR DATOS no para dar diseño a la página.

Las etiquetas principales que se utilizan para estructurar una tabla en HTML son las siguientes:

  1. table: Esta etiqueta define el inicio y el fin de la tabla en HTML. Todos los elementos de la tabla deben estar contenidos dentro de esta etiqueta.
  2. tr: Esta etiqueta define una fila en la tabla. Dentro de esta etiqueta, se deben colocar las celdas de esa fila.
  3. th: Esta etiqueta se utiliza para definir una celda de encabezado de la tabla. Las celdas de encabezado suelen contener información importante y se muestran en negrita de forma predeterminada.
  4. td: Esta etiqueta se utiliza para definir una celda de datos en la tabla. Las celdas de datos contienen la información real que se muestra en la tabla.

    2. ¿Qué etiqueta se utiliza para definir una fila en una tabla?

    Se usa la etiqueta "tr".

    3. ¿Cuál es la etiqueta utilizada para crear una celda de encabezado en una tabla?

    Se usa la etiqueta "th".

    4. ¿Cuál es la etiqueta utilizada para crear una celda de datos en una tabla?

    Se usa la etiqueta "td".

    5. ¿Qué atributo se utiliza para especificar cuántas columnas debe abarcar una celda?

    Se utiliza "colspan" seguido de un numero.

    6. ¿Qué atributo se utiliza para especificar cuántas filas debe abarcar una celda?

    Se usa "rowspan".

    7. Si necesitamos crear una tabla con 10 filas ¿qué etiqueta y cuántas veces se debe utilizar para definir las filas?

    Se usan 10 "tr" para hacer una tabla con 10 filas.
